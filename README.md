# discord-server-list

# What does it do ?
Provides information about Discord servers and shows the number of members in voice channels

# Setup

Discord bot Config file: server/src/config.json

```
1. cd server
2. npm install
3. node server

then

1. cd client
2. npm install
3. npm run dev
```
